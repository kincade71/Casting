Application for updating and sharing a podcast.

## Requirements

1. PHP 5.2+
2. Jquery-UI ~1.11.4

## Usage
- Clone repo
- Make a folder named "podcast" in the root folder
- Set your DB connection in 
``` application->config->database.php ```
